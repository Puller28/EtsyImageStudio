1. Authentication
Google or email/password login

Session management

Usage tracking per user

Use: Supabase Auth or Firebase Auth

2. Image Upload
Drag-and-drop upload (JPG/PNG, max 50MB)

Display thumbnail preview after upload

Save original in storage (Firebase/Supabase)

3. Image Processing Engine
a. Upscaling
Trigger API call to Replicate (Real-ESRGAN model)

Save upscaled result and show preview

Let user choose: 2x or 4x

b. Resizing
Automatically generate:

4x5 (8x10)

3x4 (18x24)

2x3 (12x18)

11x14

A4 (ISO)

Save each size to download package

Use: Cloudinary (transformation URLs) or on-server with sharp (Node.js)

4. Mockup Generation
User selects from 5 mockup templates

Auto-place image inside mockup scene (centered, scaled)

Add drop shadow or border if needed

Save generated mockup image

Use: Bannerbear or HTML5 Canvas overlays

5. Etsy Listing Generator
Input: short text fields:

"Artwork title/idea"

"Style keywords (e.g. watercolour, boho, Nordic)"

Output:

SEO-optimised Product Title

15 Etsy-style Tags

3-paragraph Product Description:

Paragraph 1: Describe artwork

Paragraph 2: Where to use it

Paragraph 3: What the buyer receives (e.g. 5 JPGs + mockup)

Use: OpenAI GPT-4 API (or GPT-3.5 if you want to reduce cost)
